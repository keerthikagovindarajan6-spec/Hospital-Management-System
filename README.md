# Hospital Management System

## Overview

Hospital Management System is a simple Java Swing GUI application developed for basic hospital management operations.

## Features

* Login Page
* Dashboard
* Add Patient
* View Patients
* Simple and user-friendly GUI

## Technologies Used

* Java
* Java Swing

## Project Structure

```
Hospital-Management-System
│
└── src
    ├── Main.java
    ├── Frontpage.java
    ├── Dashboard.java
    ├── patient.java
    └── ViewPatients.java
```

```

```
